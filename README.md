# L0 — 我的第一個 Creative Coding 作品

歡迎來到 Creative Code Commons！這個練習的目標，是走完一次完整流程：
**複製範本 → 改一行 → 上傳（push）→ 部署到 GitHub Pages → 貼上展示牆。**

## 你要做的事

1. 在本頁右上角按 **Use this template → Create a new repository**，命名為 `L0-你的名字`，選 **Public**，建立。
2. 把你新建的 repo `clone` 到電腦，用 VS Code 打開。
3. 打開 `sketch.js`，找到標有 **✏️** 的地方（有 5 個！），改改看：換上你的名字、調整星星數量、轉速、背景色、調色盤。把畫面拖出滑鼠、亂改一通，做出獨一無二的版本。

完成以上，你就跑過整門課的核心流程了 🎉

## 怎麼在本機預覽

在 VS Code 安裝 **Live Server** 擴充套件 → 對 `index.html` 按右鍵 → **Open with Live Server**，瀏覽器就會顯示你的作品。

---

小提醒：`sketch.js` 裡用到的 `for` 迴圈、`mouseX`、`cos()`/`sin()` 之後 L1–L2 會學到，現在照著用、先感受「改了就會變」就好。
